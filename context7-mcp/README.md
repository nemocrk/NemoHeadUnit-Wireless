# Context7 MCP Server

This is the Context7 MCP server for Cline. It provides up-to-date documentation for libraries and APIs.

## Installation

Download and install the Context7 MCP server from: https://github.com/upstash/context7-mcp

## Usage

Use the Context7 MCP server to get up-to-date documentation for any library or API you're working with.

### Example Prompts

```
Create a Next.js middleware that checks for a valid JWT in cookies
and redirects unauthenticated users to `/login`. use context7
```

```
Configure a Cloudflare Worker script to cache
JSON API responses for five minutes. use context7
```

```
Show me the Supabase auth API for email/password sign-up.
```

Context7 fetches up-to-date code examples and documentation right into your LLM's context.

## Installation Steps

1. Get a free API key at https://context7.com/dashboard
2. Configure the MCP server in cline_mcp_settings.json
3. Use the server in your prompts
